# 🌈 小雅的编程 · Xiaoya's Coding

> 一个 **6 岁小姑娘小雅** 的 **Vibe Coding** 体验 🧒💻

## 这是什么？

这是小雅（Xiaoya）—— 一个 6 岁的小姑娘 —— 用自己的话，把脑袋里天马行空的想法说出来，
和 AI（Claude Code）一起"聊着聊着就做出来"的一堆小游戏。

她不写一行代码，只负责**想点子、提要求、玩、然后说"这里我想要……"**；
AI 负责把她的话变成真正能在浏览器里玩的网页游戏。这就是所谓的 **Vibe Coding（氛围编程）**——
用聊天和想象力来创造，而不是敲代码。

每一个游戏，都是从小雅的一句童言童语开始的，比如：

- *"一个小鸟在空中飞，用鼠标躲障碍物……"*
- *"小企鹅跟姐姐玩躲猫猫，有一只手会捂住眼睛数三二一……"*
- *"一个小企鹅带着姐姐去游乐场，后来迷路了，要经过 17 个场景找回家的路……"*
- *"我要做一个逃离密室，怪物要恐怖一点，不然就不叫密室了！"*
- *"我要做一个沙威玛小店，客人会有绿到红的时间标志，还会有小偷来偷钱……"*

## ▶️ 怎么玩

**方法一：直接打开**
双击 `home.html`，用浏览器打开即可。

**方法二：本地服务器**（部分功能更稳定，例如朗读声音）

```bash
# 在项目文件夹里运行：
python3 -m http.server 8000
# 然后浏览器打开：
# http://127.0.0.1:8000/home.html
```

## 🛠️ 技术

- 纯 **HTML + CSS + JavaScript**，零依赖、零安装
- 每个游戏就是一个单独的 `.html` 文件，打开即玩
- 用到的小技术：Canvas 画图、光线投射（3D 密室）、Web Audio（音效）、
  SpeechSynthesis（朗读题目）等
- 全部由 [Claude Code](https://claude.com/claude-code) 协助完成
